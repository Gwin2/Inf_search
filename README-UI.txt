Единый WebUI для сдачи лабораторных

Запуск:
.\start-ui.bat

Альтернатива:
.\start-ui.ps1

После запуска откроется:
http://localhost:8080/index.html

Если порт 8080 занят, скрипт попробует 8081.

Для выполнения PHP-файлов нужен PHP в PATH или установленный XAMPP/OpenServer.
Для лабораторной N 4 также нужен запущенный MySQL.
